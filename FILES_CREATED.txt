## 🚀 NEXT STEPS

1. Run database setup: `createdb solar_dashboard`
2. Import schema: `psql -d solar_dashboard -f backend/src/models/database-schema.sql`
3. Install dependencies: `npm install` in both backend and frontend
4. Configure .env files
5. Start backend: `npm run dev` in backend folder
6. Start frontend: `npm start` in frontend folder
7. Access application at http://localhost:3000
