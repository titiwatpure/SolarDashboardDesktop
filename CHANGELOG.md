## v1.0.11 (2026-05-19)

### โครงการ
- สร้างรหัสโครงการอัตโนมัติ: `P{ปีพุทธ}-{ลำดับ}` (เช่น `P6905-001`)

## v1.0.10 (2026-05-19)

### ระบบเอกสาร
- แยกโฟลเดอร์ตามชื่อโครงการ + รหัส (เช่น `โซลาร์ฟาร์มบ้านนา_P001`)
- เก็บชื่อไฟล์เดิมบนดิสก์ (ไม่ใช้ UUID สุ่ม) + จัดการชื่อซ้ำ
- ดาวน์โหลดไฟล์ผ่าน native save dialog ใน Electron (ไม่ค้าง)
- เพิ่มปุ่ม "เลือก..." folder picker ในหน้าตั้งค่า (ผ่าน IPC)
- กรองเอกสารตามโครงการ (dropdown + การ์ดสรุป)
- เพิ่ม API summary จำนวนเอกสารตามโครงการ

### QA/QC
- แก้ชื่อไฟล์ว่างสำหรับ dotfiles (เช่น `.gitignore`)
- แก้ชื่อลงท้ายด้วยจุด/ช่องว่าง (Windows ไม่รองรับ)
- จำกัดความยาวชื่อไฟล์ (200), โฟลเดอร์ (100), เอกสาร (500)
- ลดโค้ดซ้ำด้วย helper function `sanitizeFolderName`
